# Release Notes – v1.0.0
NxtAbroad AI – Visa Readiness & Eligibility Intelligence Engine
Author: Ibrahim Akintunde Akinyera
Date: 2025

-------------------------------------------------------------

## Overview

This release contains the complete working version of the NxtAbroad AI Visa Readiness & Eligibility Engine. It includes the rule-based scoring system, input and output datasets, architecture diagrams, evidence documentation, and codebase used as Evidence 1 – Mandatory Criterion (Innovation) for the UK Global Talent Visa (Digital Technology).

The engine evaluates academic strength, financial readiness, documentation quality, and credibility indicators to produce a Visa Readiness Score for international student recruitment.

-------------------------------------------------------------

## What’s Included in v1.0.0

1. Core Engine Logic
   - src/rules_engine.py – Implements academic, financial, documentation, and credibility rules.
   - src/scorer.py – Aggregates rule outputs and generates final readiness scores.
   - src/utils.py – Helper functions for CSV parsing, validation, and normalisation.

2. Sample Data
   - data/raw/sample_profiles.csv – Synthetic visa-style applicant profiles.
   - data/processed/readiness_report.csv – Sample scoring results generated by the engine.

3. Documentation
   - README.md – Full technical overview of the system.
   - TECH_NATION_EVIDENCE.md – Formal mapping to the Mandatory Criterion.
   - docs/figures/architecture.png – System architecture.
   - docs/figures/scoring_flow.png – Scoring flow diagram.

4. Dependencies
   - requirements.txt – Python dependency list.
   - LICENSE – MIT license.

## How to Run

Run the following command to process the sample data:

python src/scorer.py \
    --input data/raw/sample_profiles.csv \
    --output data/processed/readiness_report.csv

This will evaluate all sample applicants and generate the readiness scoring output.
